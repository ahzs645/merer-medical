Mere Medical currently has integration support for the following patient portal and FHIR ecosystems:

- MyChart/Epic
- Cerner Health
- Veradigm and Allscripts/FollowMyHealth-style FHIR endpoints
- DrChrono/OnPatient
- Healow/eClinicalWorks
- VA sandbox access

Production availability depends on each portal's developer approval process, OAuth configuration, and whether the portal exposes patient-access FHIR endpoints. Some integrations may be easier to test in sandbox than production.

Support for more patient portals is coming soon.

Mere Medical is still in active development. If you run into any issues when connecting using a supported portal, [please submit a bug report](mailto:cfu288@meremedical.co).
